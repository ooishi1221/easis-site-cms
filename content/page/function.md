## イベント効果測定のサービス内容

イベント効果測定は、展示会出展担当者様やイベント開催担当者様のご負担軽減とイベント運営戦略などに活用されています。また、本サービスが企業様に選ばれる理由として、以下の特徴が挙げられます。

**1.AI分析カメラを用いた自動測定  
**測定スタッフを配置する必要はなく、展示会やイベントの出展ブースにAI分析カメラを設置するだけで自動測定される手軽さ  
  
**2.これまで得られなかった情報の入手  
**これまでイベント出展の成果をリード獲得数、アポ成立数、契約数だけで判断していた企業様が気になされていたブース内訪問数やブース隣接通路の流入数などが取得可能  
  
**3.イベント終了後に分析内容をレポート化  
**自動測定されたデータと分析スタッフによって、イベント終了後にご依頼を承ったイベントに関する分析、改善提案等を取り纏めてレポート化

また、本ページでは、その全てを紹介することは出来ませんが、1小間で展示会に出展した際の主な測定内容などを8個だけ紹介いたします。

### ネットワークカメラサービス「キヅクモ」を用いたイベント効果測定

![](https://easis.wit-one.co.jp/wp-content/uploads/2025/10/fun01.jpg)

ネットワークカメラサービス「キヅクモ」につきましては、弊社で代理店販売を行っております。

### 1：出展環境に合わせた測定

![](https://easis.wit-one.co.jp/wp-content/uploads/2025/10/fun001.jpg)

説明

出展ブースの大きさや位置に合わせて測定内容を協議いたします。  
掲載内容はブース内占有面積が少なかったことから通路をメインとして、流入数と占有率を測定しております。

### 2：通路流入測定（トータル）

![](https://easis.wit-one.co.jp/wp-content/uploads/2025/10/fun02.jpg)

説明

出展ブース前通路における左右からの来場者の流入数を測定。  
全開催期間のトータル流入数やデータに基づく分析内容を掲載しております。

### 3：通路流入測定（開催3日間）

![](https://easis.wit-one.co.jp/wp-content/uploads/2025/10/fun03.jpg)

説明

出展ブース前通路における左右からの来場者の流入数を測定。  
開催日毎の流入数を掲載しております。

### 4：通路流入測定（各日）

![](https://easis.wit-one.co.jp/wp-content/uploads/2025/10/fun04.jpg)

説明

出展ブース前通路における左右からの来場者の流入数を測定。  
開催日毎の時間別流入推移を掲載しております。

### 5：占有率測定（トータル）

![](https://easis.wit-one.co.jp/wp-content/uploads/2025/10/fun05.jpg)

説明

出展ブース前通路における来場者の占有率（滞在状況）を測定。  
全開催期間を通しての最大ピーク占有率や平均占有率、データに基づく分析内容を掲載しております。

### 6：占有率測定（開催3日間）

![](https://easis.wit-one.co.jp/wp-content/uploads/2025/10/fun06.jpg)

説明

出展ブース前通路における来場者の占有率（滞在状況）を測定。  
開催日毎のピーク占有率や平均占有率を掲載しております。

### 7：占有率測定（各日）

![](https://easis.wit-one.co.jp/wp-content/uploads/2025/10/fun07.jpg)

説明

出展ブース前通路における来場者の占有率（滞在状況）を測定。  
開催日毎の時間別ピーク占有率や平均占有率推移を掲載しております。

### 8：出展改善提案

![](https://easis.wit-one.co.jp/wp-content/uploads/2025/10/fun08.jpg)

説明

各測定データ及び分析スタッフによる周囲の出展企業ブースの調査結果に基づき、イベント終了後に分析レポートを納品いたします。  
出展担当者様の社内向け説明や次回出展戦略にご活用いただけます。
